# Market Range Viewer

一个专门解决“股票 App 时间范围不够自由”的网页应用，支持手机和电脑访问。

## 已实现能力

- 搜索或直接输入全球常见行情代码
- 支持股票、港股、马股、外汇、加密货币、大宗商品
- `分时 / 日K / 周K / 月K`
- `1M / 3M / 6M / 1Y / 3Y / 5Y / MAX` 预设区间
- 自定义开始日期和结束日期
- 图表拖拽、缩放、滚轮缩放
- MA5 / MA20 / MA60
- 成交量开关
- 本地自选股

## 常见代码示例

- 美股: `AAPL`
- 港股: `0700.HK`
- 马股: `5183.KL`
- 外汇: `EURUSD=X`
- 黄金期货: `GC=F`
- 比特币: `BTC-USD`

## 本地运行

```bash
npm install
npm run dev
```

打开 `http://localhost:3000`

## 生产构建

```bash
npm run build
npm run start
```

## 上线部署

最方便的方式是部署到 Vercel：

1. 把项目推到 GitHub
2. 在 Vercel 导入这个仓库
3. Framework 选择 Next.js
4. 直接部署

部署后你就可以在电脑和手机上通过同一个网址访问。

## 数据说明

项目使用服务端接口代理免费公开行情数据，浏览器不直接请求第三方源，方便后续上线部署。
